# 🌦️ Weather App

Aplicación web interactiva que permite consultar el clima de cualquier ciudad en tiempo real utilizando una API externa. El proyecto está desarrollado con **HTML, CSS y JavaScript puro**, enfocándose en una interfaz moderna y una experiencia de usuario fluida.

---

## 🚀 Características

* 🔍 Búsqueda de ciudades en tiempo real
* 🌡️ Muestra temperatura actual
* ☁️ Descripción del clima
* 💧 Información adicional (humedad, viento, etc.)
* 🌙 Modo oscuro automático y manual
* ✨ Animaciones suaves y transiciones
* 📱 Diseño responsive

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox, Glassmorphism, Animaciones)
* JavaScript (Fetch API, DOM, Eventos)
* API de clima (OpenWeather)

---

## 🎨 UI / UX

El diseño está basado en tendencias modernas como:

* Glassmorphism (efecto vidrio)
* Transiciones suaves entre modo claro y oscuro
* Animaciones de entrada y actualización de contenido
* Interacciones dinámicas (hover, click)

## ⚙️ Cómo usar

1. Clona el repositorio:

```id="k8v2x1"
git clone https://github.com/tu-usuario/weather-app.git
```

2. Abre el archivo `index.html` en tu navegador.

3. Ingresa una ciudad y presiona **Buscar** o `Enter`.

---

## 🔑 Configuración

Este proyecto utiliza una API Key de OpenWeather.

Debes reemplazar en el archivo JavaScript:

```js id="u2d8s9"
const API_KEY = "TU_API_KEY";
```

Puedes obtener una clave gratuita en:
https://openweathermap.org/
